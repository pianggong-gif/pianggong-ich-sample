平江方言非遗数据库 Metadata 规范

Pingjiang Dialect Intangible Heritage Metadata Schema / Version 2.0

0. 项目定位

本规范用于整理平江方言相关非物质文化遗产资料，包括影戏、灯戏、夜歌、唱经、三花白、吟诵、快板、歌谣、童谣、方言对话、古书诵读，以及手抄本、科仪本、戏本等文献资料。

核心原则：多模态 + 多语言层 + AI-ready
