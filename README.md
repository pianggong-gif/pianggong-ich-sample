# PG_TY_001_banzhy Bilibili + EAF Web Demo

This package displays the Pingjiang dialect nursery rhyme "斑鸠" with:

- Bilibili embedded video
- WAV audio player
- SRT subtitle table
- EAF tier display
- Four-layer aligned annotation display
- metadata JSON
